Ejercicios Desarrollados

Ejercicio 1: Resolución de Ecuación Cuadrática

Resuelve la ecuación cuadrática: \(2x^2 - 4x - 6 = 0\)

1 Identificamos los coeficientes: a = 2, b = -4, c = -6
2 Aplicamos la fórmula cuadrática: \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
3 Sustituimos los valores: \(x = \frac{-(-4) \pm \sqrt{(-4)^2 - 4(2)(-6)}}{2(2)} = \frac{4 \pm \sqrt{16 + 48}}{4} = \frac{4 \pm \sqrt{64}}{4} = \frac{4 \pm 8}{4}\)
4 Calculamos las dos soluciones: \(x_1 = \frac{4 + 8}{4} = \frac{12}{4} = 3\) \(x_2 = \frac{4 - 8}{4} = \frac{-4}{4} = -1\)

Ejercicio 2: Función Lineal y Gráfica

Dada la función \(f(x) = 2x - 3\), encuentra la pendiente, el intercepto con el eje y, y grafica la función.

1 La función está en forma pendiente-intercepto: \(y = mx + b\)
2 Pendiente (m) = 2
3 Intercepto con el eje y (b) = -3 → Punto (0, -3)
4 Para graficar, comenzamos en el punto (0, -3) y usamos la pendiente: por cada aumento de 1 en x, y aumenta 2.

Ejercicio 3: Simplificación de Expresiones Algebraicas

Simplifica la expresión: \(\frac{3x^2 - 12}{x^2 + 4x + 4} \cdot \frac{x+2}{x-2}\)

1 Factorizamos: \(3x^2 - 12 = 3(x-2)(x+2)\), \(x^2 + 4x + 4 = (x+2)^2\)
2 Reescribimos: \(\frac{3(x-2)(x+2)}{(x+2)^2} \cdot \frac{x+2}{x-2}\)
3 Simplificamos: \(\frac{3 \cancel{(x-2)} \cancel{(x+2)}}{\cancel{(x+2)} \cancel{(x+2)}} \cdot \frac{\cancel{x+2}}{\cancel{x-2}} = 3\)

Ejercicio 4: Resolución de Sistema de Ecuaciones

Resuelve el sistema de ecuaciones: \(2x + y = 7\) y \(3x - 2y = 0\)

1 Despejamos y en la primera ecuación: \(y = 7 - 2x\)
2 Sustituimos en la segunda: \(3x - 2(7 - 2x) = 0\)
3 Resolvemos: \(3x - 14 + 4x = 0 \Rightarrow 7x = 14 \Rightarrow x = 2\)
4 Sustituimos x = 2: \(y = 7 - 2(2) = 3\)

Ejercicio 5: Problema de Aplicación - Optimización

Se quiere construir una caja rectangular sin tapa con una base cuadrada. Si el área total es de 300 cm², ¿qué dimensiones maximizan el volumen?

1 Sea x el lado de la base y h la altura: \(x^2 + 4xh = 300\)
2 Despejamos h: \(h = \frac{300 - x^2}{4x}\)
3 Volumen: \(V = x^2 \cdot h = \frac{300x - x^3}{4}\)
4 Derivamos: \(V' = \frac{300 - 3x^2}{4} = 0 \Rightarrow x = 10\)
5 Calculamos h: \(h = \frac{300 - 100}{40} = 5\)

Ejercicios Desarrollados

Ejercicio 1: Cálculo de Límites

Calcula el límite: \(\lim_{x \to 2} \frac{x^2 - 4}{x - 2}\)

1 Factorizamos el numerador: \(x^2 - 4 = (x-2)(x+2)\)
2 Simplificamos: \(\frac{(x-2)(x+2)}{x-2} = x+2\)
3 Evaluamos el límite: \(\lim_{x \to 2} (x+2) = 4\)

Ejercicio 2: Derivación Implícita

Encuentra \(\frac{dy}{dx}\) para la ecuación \(x^2 + y^2 = 25\)

1 Derivamos ambos lados: \(\frac{d}{dx}(x^2) + \frac{d}{dx}(y^2) = \frac{d}{dx}(25)\)
2 Aplicamos reglas de derivación: \(2x + 2y\frac{dy}{dx} = 0\)
3 Despejamos \(\frac{dy}{dx}\): \(2y\frac{dy}{dx} = -2x\)
4 Obtenemos: \(\frac{dy}{dx} = -\frac{x}{y}\)

Ejercicio 3: Optimización

Encuentra dos números positivos cuya suma sea 50 y cuyo producto sea máximo.

1 Sean x y y los números: \(x + y = 50\), queremos maximizar \(P = x \cdot y\)
2 Expresamos y en términos de x: \(y = 50 - x\)
3 Producto: \(P = x(50 - x) = 50x - x^2\)
4 Derivamos: \(P' = 50 - 2x\)
5 Igualamos a cero: \(50 - 2x = 0 \Rightarrow x = 25\)
6 Entonces \(y = 25\), y el producto máximo es \(25 \times 25 = 625\)

Ejercicio 4: Integral Definida

Calcula \(\int_0^1 (3x^2 + 2x + 1) dx\)

1 Encontramos la antiderivada: \(\int (3x^2 + 2x + 1) dx = x^3 + x^2 + x + C\)
2 Evaluamos en los límites: \([1^3 + 1^2 + 1] - [0^3 + 0^2 + 0] = 3 - 0 = 3\)

Ejercicio 5: Regla de L'Hôpital

Calcula \(\lim_{x \to 0} \frac{\sin x}{x}\)

1 Verificamos que es una indeterminación 0/0: \(\sin 0 = 0\), \(0 = 0\)
2 Aplicamos L'Hôpital: derivamos numerador y denominador
3 \(\frac{d}{dx}(\sin x) = \cos x\), \(\frac{d}{dx}(x) = 1\)
4 El límite es \(\lim_{x \to 0} \frac{\cos x}{1} = \cos 0 = 1\)

Ejercicios Desarrollados

Ejercicio 1: Circuito Rectificador de Media Onda

Diseña un circuito rectificador de media onda con diodo para una señal de entrada de 12V RMS. Calcula el voltaje de salida promedio.

1 Voltaje pico de entrada: \(V_p = V_{RMS} \times \sqrt{2} = 12 \times 1.414 = 17V\)
2 Considerando la caída en el diodo (0.7V para silicio): \(V_{out(pico)} = 17 - 0.7 = 16.3V\)
3 Voltaje promedio de salida: \(V_{avg} = \frac{V_p}{\pi} = \frac{16.3}{3.1416} = 5.19V\)

Ejercicio 2: Polarización de Transistor BJT

Calcula los puntos de operación (ICQ, VCEQ) para un transistor BJT con polarización por divisor de voltaje. Datos: VCC=12V, R1=22kΩ, R2=10kΩ, RC=2.2kΩ, RE=1kΩ, β=100.

1 Voltaje en la base: \(V_B = V_{CC} \times \frac{R2}{R1+R2} = 12 \times \frac{10}{32} = 3.75V\)
2 Voltaje en el emisor: \(V_E = V_B - V_{BE} = 3.75 - 0.7 = 3.05V\)
3 Corriente en el emisor: \(I_E = \frac{V_E}{R_E} = \frac{3.05}{1000} = 3.05mA\)
4 \(I_C ≈ I_E = 3.05mA\)
5 \(V_{CE} = V_{CC} - I_C(R_C + R_E) = 12 - 0.00305(2200 + 1000) = 12 - 9.76 = 2.24V\)

Ejercicio 3: Amplificador Operacional No Inversor

Diseña un amplificador operacional no inversor con ganancia de 10. Si el voltaje de entrada es 0.5V, ¿cuál es el voltaje de salida?

1 Fórmula de ganancia: \(A = 1 + \frac{R_f}{R_1} = 10\)
2 Elegimos R1 = 10kΩ, entonces \(\frac{R_f}{10k} = 9 \Rightarrow R_f = 90kΩ\)
3 \(V_{out} = A \times V_{in} = 10 \times 0.5 = 5V\)

Ejercicio 4: Filtro Pasa-Bajos RC

Diseña un filtro pasa-bajos con frecuencia de corte de 1kHz. Calcula la atenuación a 5kHz.

1 Frecuencia de corte: \(f_c = \frac{1}{2\pi RC} = 1000Hz\)
2 Elegimos C = 10nF, entonces \(R = \frac{1}{2\pi \times 1000 \times 10^{-8}} = 15.9kΩ\)
3 Atenuación a 5kHz: \(\frac{V_{out}}{V_{in}} = \frac{1}{\sqrt{1 + (f/f_c)^2}} = \frac{1}{\sqrt{1 + 25}} = 0.196\)
4 Atenuación en dB: \(20\log(0.196) = -14.15dB\)

Ejercicio 5: Regulador de Voltaje con Zener

Diseña un regulador de voltaje con diodo Zener de 5.6V para una carga de 1kΩ y voltaje de entrada entre 8-12V.

1 Corriente máxima de carga: \(I_L = \frac{5.6}{1000} = 5.6mA\)
2 Asumimos corriente mínima en Zener: \(I_{Zmin} = 5mA\)
3 Corriente total: \(I_T = I_Z + I_L = 5 + 5.6 = 10.6mA\)
4 Resistencia en serie: \(R_S = \frac{V_{inmin} - V_Z}{I_T} = \frac{8 - 5.6}{0.0106} = 226Ω\)
5 Usamos resistencia estándar de 220Ω

Ejercicios Desarrollados

Ejercicio 1: Probabilidad Condicional

En una clase, el 60% de los estudiantes son mujeres y el 40% son hombres. El 30% de las mujeres y el 20% de los hombres tienen beca. Si se selecciona un estudiante al azar y tiene beca, ¿cuál es la probabilidad de que sea mujer?

1 Definimos eventos: M = ser mujer, H = ser hombre, B = tener beca
2 P(M) = 0.6, P(H) = 0.4, P(B|M) = 0.3, P(B|H) = 0.2
3 Aplicamos teorema de Bayes: P(M|B) = [P(B|M)P(M)] / [P(B|M)P(M) + P(B|H)P(H)]
4 Calculamos: P(M|B) = [0.3*0.6] / [0.3*0.6 + 0.2*0.4] = 0.18 / (0.18+0.08) = 0.18/0.26 ≈ 0.6923
5 La probabilidad es aproximadamente 69.23%

Ejercicio 2: Distribución Binomial

La probabilidad de que un componente falle es 0.1. Si se prueban 10 componentes, ¿cuál es la probabilidad de que fallen exactamente 3?

1 Es una distribución binomial con n=10, p=0.1, x=3
2 Fórmula: P(X=3) = C(10,3) * (0.1)^3 * (0.9)^7
3 C(10,3) = 120
4 P(X=3) = 120 * 0.001 * 0.4783 ≈ 0.0574
5 La probabilidad es aproximadamente 5.74%

Ejercicio 3: Distribución Normal

Las calificaciones de un examen siguen una distribución normal con media 75 y desviación estándar 10. ¿Qué porcentaje de estudiantes obtuvo entre 70 y 85 puntos?

1 Convertimos a puntuaciones z: z1 = (70-75)/10 = -0.5, z2 = (85-75)/10 = 1.0
2 Buscamos en tabla: P(Z < -0.5)=0.3085, P(Z < 1.0)=0.8413
3 P(-0.5 < Z < 1.0)=0.8413 - 0.3085=0.5328
4 El porcentaje es 53.28%

Ejercicio 4: Valor Esperado

En un juego se lanza un dado. Si sale par, ganas Q20; si sale 1 o 3, pierdes Q30; si sale 5, ganas Q50. ¿Cuál es el valor esperado de este juego?

1 Probabilidades: P(par) = 3/6 = 0.5, P(1 o 3) = 2/6 ≈ 0.333, P(5) = 1/6 ≈ 0.167
2 Valores: V(par) = 20, V(1 o 3) = -30, V(5) = 50
3 Valor esperado: E = (0.5*20) + (0.333*(-30)) + (0.167*50)

Explica la Ley de los Grandes Números y proporciona un ejemplo de su aplicación.

3 Aplicación: En seguros, permite predecir siniestros basándose en grandes volúmenes de datos.

Se desea estimar la media de una población normal con varianza desconocida. Una muestra de 25 elementos tiene media 80 y desviación estándar 12. Construye un intervalo de confianza del 95% para la media poblacional.

2 Estadístico de prueba: t = (x̄ - μ) / (s/√n) = (980-1000)/(80/√30) = -20/14.61 ≈ -1.368

Ejercicio 3: Regresión Lineal

Ajusta una recta de regresión para los datos: x:1,2,3,4,5 y y:2,4,5,4,5. Calcula el coeficiente de correlación.

1 Calculamos medias: x̄ = 3, ȳ = 4
2 Calculamos Sxy = Σ(x-x̄)(y-ȳ) = 4, Sxx = Σ(x-x̄)² = 10
3 Pendiente: b = Sxy/Sxx = 4/10 = 0.4

Ejercicio 4: ANOVA de Una Vía

1 Calculamos medias: x̄₁ = 75, x̄₂ = 80, x̄₃ = 85, media total = 80
2 SCT = ΣΣ(xij - x̄)² = 300, SCE = Σnᵢ(x̄ᵢ - x̄)² = 150, SCR = SCT - SCE = 150

1 λ = 10 clientes/hora, μ = 15 clientes/hora
2 Factor de utilización: ρ = λ/μ = 10/15 = 0.6667
3 Probabilidad de 0 clientes: P₀ = 1 - ρ = 1 - 0.6667 = 0.3333
4 Número promedio en el sistema: L = λ/(μ-λ) = 10/(15-10) = 2 clientes
5 Tiempo promedio en el sistema: W = 1/(μ-λ) = 1/5 = 0.2 horas = 12 minutos

Ejercicio 5: Programación Dinámica

Resuelve el problema de la mochila: capacidad=5kg, items: A(2kg, Q3), B(3kg, Q4), C(4kg, Q5), D(1kg, Q2). Maximiza el valor total.

1 Creamos tabla para capacidades de 0 a 5 kg
2 Para cada capacidad, evaluamos incluir cada item que cabe
3 Solución: Incluir B (3kg, Q4) y D (1kg, Q2) con peso total 4kg y valor Q6
4 O incluir A (2kg, Q3) y C (4kg, Q5) con peso total 6kg (no válido)
5 O incluir A, B y D: 2+3+1=6kg (no válido)
6 Mejor solución válida: B y D con valor Q6

Ejercicios Desarrollados

Ejercicio 1: Movimiento Parabólico

Se lanza un proyectil con velocidad inicial de 50 m/s con un ángulo de 37° sobre la horizontal. Calcula: a) Altura máxima, b) Alcance horizontal, c) Tiempo de vuelo.

1 Componentes de velocidad: v₀ₓ = 50*cos37° ≈ 50*0.8 = 40 m/s, v₀y = 50*sin37° ≈ 50*0.6 = 30 m/s
2 Altura máxima: y_max = (v₀y²)/(2g) = (30²)/(2*9.8) = 900/19.6 ≈ 45.92 m
3 Tiempo de vuelo: t_v = (2*v₀y)/g = (2*30)/9.8 ≈ 6.12 s
4 Alcance: x_max = v₀ₓ * t_v = 40 * 6.12 ≈ 244.8 m

Ejercicio 2: Leyes de Newton

Un bloque de 10 kg es arrastrado sobre una superficie horizontal con una fuerza de 50 N que forma un ángulo de 30° con la horizontal. Si el coeficiente de fricción cinética es 0.2, calcula la aceleración del bloque.

1 Componentes de la fuerza: Fₓ = 50*cos30° ≈ 43.3 N, Fy = 50*sin30° = 25 N
2 Fuerza normal: N = mg - Fy = 10*9.8 - 25 = 98 - 25 = 73 N
3 Fuerza de fricción: f = μ*N = 0.2*73 = 14.6 N
4 Fuerza neta horizontal: F_net = Fₓ - f = 43.3 - 14.6 = 28.7 N
5 Aceleración: a = F_net/m = 28.7/10 = 2.87 m/s²

Ejercicio 3: Conservación de Energía

Un bloque de 2 kg se desliza desde el reposo desde una altura de 5 m por una rampa sin fricción. Luego recorre una superficie horizontal con fricción (μ=0.3) hasta detenerse. ¿Qué distancia recorre en la superficie horizontal?

1 Energía potencial inicial: Ep = mgh = 2*9.8*5 = 98 J
2 Esta energía se convierte en energía cinética al final de la rampa: Ec = 98 J
3 En la superficie horizontal, el trabajo de fricción disipa esta energía: W_f = f*d = μ*N*d = μ*m*g*d
4 Igualamos: μ*m*g*d = 98 J
5 Despejamos d: d = 98/(μ*m*g) = 98/(0.3*2*9.8) = 98/5.88 ≈ 16.67 m

Ejercicio 4: Momento de Inercia

Calcula el momento de inercia de una varilla delgada de masa M y longitud L que gira alrededor de un eje perpendicular que pasa por su centro.

1 Densidad lineal: λ = M/L
2 Consideramos un elemento de masa dm = λ*dx a distancia x del centro
3 Momento de inercia del elemento: dI = x²*dm = x²*λ*dx
4 Integramos desde -L/2 hasta L/2: I = ∫x²*λ*dx = λ*[x³/3] de -L/2 a L/2
5 I = λ*(L³/24 + L³/24) = λ*(L³/12) = (M/L)*(L³/12) = (1/12)ML²

Ejercicio 5: Primera Ley de la Termodinámica

Un gas ideal experimenta una expansión isotérmica a 300 K desde un volumen de 2 L a 5 L. Si el gas tenía inicialmente 2 moles, calcula el trabajo realizado y el calor transferido.

1 En proceso isotérmico, ΔU = 0, por tanto Q = W
2 Trabajo en expansión isotérmica: W = nRT*ln(Vf/Vi)
3 W = 2*8.314*300*ln(5/2) = 4988.4*ln(2.5) ≈ 4988.4*0.9163 ≈ 4570 J
4 Calor transferido: Q = W = 4570 J

Ejercicios Desarrollados

Ejercicio 1: Ley de Coulomb

Dos cargas puntuales de 3 μC y -4 μC están separadas 50 cm en el vacío. Calcula la fuerza entre ellas y determina si es de atracción o repulsión.

1 Ley de Coulomb: F = k*(|q₁*q₂|)/r², k = 9×10⁹ Nm²/C²
2 F = (9×10⁹)*|3×10⁻⁶ * (-4)×10⁻⁶|/(0.5)² = (9×10⁹)*12×10⁻¹²/0.25
3 F = (9×10⁹)*4.8×10⁻¹¹ = 4.32×10⁻¹ N = 0.432 N
4 Como las cargas son de signo opuesto, la fuerza es de atracción

Ejercicio 2: Campo Eléctrico

Calcula el campo eléctrico en el punto medio entre dos cargas: q₁=2×10⁻⁶ C en (0,0) y q₂=-3×10⁻⁶ C en (4,0) metros.

1 Punto medio: (2,0)
2 Distancia de cada carga al punto: r = 2 m
3 Campo de q₁: E₁ = k*q₁/r² = (9×10⁹)*(2×10⁻⁶)/4 = 4500 N/C (dirigido hacia +x)
4 Campo de q₂: E₂ = k*|q₂|/r² = (9×10⁹)*(3×10⁻⁶)/4 = 6750 N/C (dirigido hacia +x porque atrae carga positiva)
5 Campo total: E = E₁ + E₂ = 4500 + 6750 = 11250 N/C en dirección +x

Ejercicio 3: Ley de Ampère

Un alambre largo y recto lleva una corriente de 5 A. Calcula el campo magnético a una distancia de 10 cm del alambre.

1 Ley de Ampère: B = (μ₀*I)/(2π*r), μ₀ = 4π×10⁻⁷ T*m/A
2 B = (4π×10⁻⁷ * 5)/(2π*0.1) = (2×10⁻⁶)/(0.1) = 2×10⁻⁵ T
3 La dirección es tangente a círculos concéntricos alrededor del alambre

Ejercicio 4: Reflexión y Refracción

Un rayo de luz incide desde el aire (n=1) sobre un vidrio (n=1.5) con un ángulo de 30°. Calcula el ángulo de refracción y el ángulo crítico para reflexión total.

1 Ley de Snell: n₁*sinθ₁ = n₂*sinθ₂
2 1*sin30° = 1.5*sinθ₂ → 0.5 = 1.5*sinθ₂ → sinθ₂ = 0.5/1.5 ≈ 0.3333
3 θ₂ = arcsin(0.3333) ≈ 19.47°
4 Ángulo crítico: sinθ_c = n₂/n₁ = 1/1.5 ≈ 0.6667 → θ_c = arcsin(0.6667) ≈ 41.81°

Ejercicio 5: Efecto Fotoeléctrico

La función trabajo del sodio es 2.3 eV. ¿Cuál es la longitud de onda máxima que puede causar emisión fotoeléctrica? Si se usa luz de 200 nm, ¿cuál es la energía cinética máxima de los electrones emitidos?

1 Energía umbral: E_0 = 2.3 eV = 2.3 * 1.6×10⁻¹⁹ = 3.68×10⁻¹⁹ J
2 Longitud de onda máxima: λ_max = h*c/E_0
3 h = 6.63×10⁻³⁴ J*s, c = 3×10⁸ m/s → h*c = 1.989×10⁻²⁵ J*m
4 λ_max = 1.989×10⁻²⁵ / 3.68×10⁻¹⁹ ≈ 5.404×10⁻⁷ m = 540.4 nm
5 Para λ=200 nm: E = h*c/λ = 1.989×10⁻²⁵ / 200×10⁻⁹ = 9.945×10⁻¹⁹ J
6 Energía cinética: K_max = E - E_0 = 9.945×10⁻¹⁹ - 3.68×10⁻¹⁹ = 6.265×10⁻¹⁹ J ≈ 3.92 eV
